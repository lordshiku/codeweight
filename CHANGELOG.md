# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - initial release

- Multi-language analysis for Python, JavaScript/TypeScript, Rust, and Java
- Cyclomatic complexity estimation per file and function
- Maintainability scoring (0–100) based on complexity, nesting, and file length
- JSON output mode for scripting and CI integration
- Colored table output for terminal reports
- GitHub Actions CI workflow (tests and clippy)
